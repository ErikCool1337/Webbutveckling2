1a. Uppgifter:
  1. Varför används storyboard?: En enkel prototyp av webbplattsen med fokus på funktionalitet istället för grafik. Man kan kalla storyboards för webbplatsens riktning.
  2. Varför används wireframe?: wireframe är en grov skiss eller struktur av en webbplatts
  3. ? Vilken metod kan användas när en klient frågar efter en resurs?: GET-metoden är när en klient skickar en GET förfrågan. Servern svarar med den begärda resursen om den finns tillgänglig.
    POST-metoden: Denna metod används för att skicka data till servern för att behandlas och lagras. Det kan vara användbart när klienten vill skicka in data, till exempel när man fyller i ett webbformulär.
  4. Vad är det som är skillnad mellan HEAD och GET metod?:
    HEAD - Samma som get men får bara svaret som innehåller http huvudet
    GET: servern svarar med den begärda resursen om den finns tillgänglig.
  5. Pa vilket sätt genomförs “Hallway testning”? Slumpmässiga individer används för att testa webbplatsen.
  6. grafisk designer -> "look and feel". FrontEnd programmerare -> HTML- och CSS-dokument. Användarupplevelsedesigner -> Storyboards. Backend-programmerare -> PHP-skript.
  7. Vad gör W3C? World Wide Web Consortium, är en internationell organisation som ansvarar för att utveckla och fastställa webbstandarder och riktlinjer. De arbetar med att definiera och främja öppna webbstandarder som HTML, CSS och JavaScript. W3C spelar en viktig roll i att säkerställa att webben förblir öppen, interoperabel och tillgänglig för alla användare oavsett vilken webbläsare eller enhet de använder.
  8. html -> Identifierar text som en rubrik på andra nivån (h2). CSS -> Gör alla rubriker på andra nivån (h2) blå. JavaScript -> Kontrollerar ett formulärfält för en giltig inmatning. Ruby -> Skapar en anpassad webbapplikation på serversidan
  9. Frontend webbutveckling fokuserar på det som användaren ser och interagerar med på webbplatsen. Frontend-utvecklare arbetar med HTML, CSS och JavaScript för att skapa användargränssnittet, layouten och interaktionerna. De fokuserar på att skapa en bra användarupplevelse.
    Backend webbutveckling är ansvarig för den "bakom kulisserna" delen av webbplatsen. Backend-utvecklare arbetar med servern, databaser och serverapplikationer. De hanterar datalagring, hantering av användaruppgifter, och att säkerställa att webbapplikationen fungerar korrekt och säkert. Backend-utvecklare fokuserar på att göra webbplatsen funktionell och hantera data och logik.
  10. Ett FTP-verktyg (File Transfer Protocol) används för att överföra filer mellan din lokala dator och en fjärrserver, vanligtvis för att publicera en webbplats eller överföra filer till en webbserver. Med ett FTP-verktyg kan du enkelt ladda upp och hämta filer från servern. För att få ett FTP-verktyg kan du ladda ner och installera det från internet. Det finns många gratis och kommersiella FTP-klientprogram tillgängliga för olika operativsystem. Några exempel på populära FTP-klienter inkluderar FileZilla, Cyberduck och WinSCP.
